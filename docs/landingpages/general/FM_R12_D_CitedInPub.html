FAIR Metric - R1.2 - Data - cited in publication

This metric evaluates whether there is evidence that the research object identifier(s) linked to the metadata identifier provided to the evaluation is cited in one or more scholarly publication repositories, providing one facet of evaluation for this R1.2.

R1.2 requires that a research object be associated with detailed provenance. The purpose of this principle is to ensure that connections between a research object and related scholarly outputs are documented, demonstrating that the object is being reused, cited, or otherwise referenced in the research literature. This metric evaluates whether there is evidence that the research object identifier provided to the evaluation is cited in one or more scholarly publication repositories. Under this interpretation, the evaluation examines external citation databases or services (specifically, OpenAlex and OpenCitations) for references to the research object DOI in publication metadata, reference lists, or full text. A resource satisfies this metric if at least one scholarly publication can be found in either OpenAlex and/or OpenCitations that explicitly cites or references the research object identified by the input identifier. The presence of one or more such citation is interpreted as a structured link from the data to the scholarly literature, supporting provenance and reuse context.

# R1.3 (empty of gen3) There is currently no plan to provide a generic metric for R1.3, due to the inherent community-specific nature of it.
