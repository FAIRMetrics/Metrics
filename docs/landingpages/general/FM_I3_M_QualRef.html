FAIR Metric - I3 - Metadata - Contains Qualified Outward Links

This metric evaluates whether there are qualified (labelled or named) links in the metadata that reference external IRIs; specifically, IRIs that do not share the same domain name as that of the provided identifier.



I3 requires that metadata include qualified references to other metadata. The purpose of this principle is to ensure that digital objects are connected to the wider research ecosystem through explicitly defined and semantically meaningful relationships. This metric evaluates whether there are qualified (labelled or named) links in the metadata that reference external IRIs. Under this interpretation, the evaluation retrieves the metadata upon resolution of the provided identifier and identifies all labelled relationships that reference IRIs. The domain names of the referenced IRIs are then compared against the domain name of the identifier provided to the evaluation tool. Links that resolve to IRIs with a different domain name are considered external references. Only qualified relationships — that is, links expressed with an explicit relationship type — are considered. A resource satisfies this metric if at least one qualified reference to an external IRI (i.e., with a different domain name than the identifier under evaluation) is present in the metadata.
