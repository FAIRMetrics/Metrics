FAIR Metric - F3 - Metadata - includes its own identifier

This metric evaluates whether the metadata record includes its own identifier, distinct from the identifier assigned to the research object it describes.



This metric is concerned with the record-level traceability aspect of FAIR; while the F3 principle was originally created to require only the research object identifier within the metadata record, this metric expands that requirement to ensure that the metadata also includes its own identifier. This ensures that the metadata itself is machine-actionable and can be retrieved, accessed, and managed as an independent entity in an indexing system. If a metadata record lacks its own unique, resolvable identifier, it cannot be reliably cited, tracked, or versioned independently of the research object it describes. This metric evaluates whether the metadata resolved from the identifier provided as input contains that identifier.
