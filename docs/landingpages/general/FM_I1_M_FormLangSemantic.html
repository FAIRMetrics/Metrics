FAIR Metric - I1 - Metadata - Uses Knowledge representation language (semantic)

This metric evaluates whether there is a resolved metadata representation that conforms to a recognised semantic knowledge representation language.



I1 requires that metadata use a formal, accessible, shared and broadly applicable language for knowledge representation. The purpose of this principle is to ensure that metadata, when retrieved, is expressed in a formally defined knowledge representation language. In this metric, the definition of syntactic and semantic formats follows the [FAIRsharing guidance](https://fairsharing.gitbook.io/fairsharing/about-our-records/fair-assistance/syntactic-vs-semantic-knowledge-representation-languages). Under this semantic interpretation, a resource satisfies this metric if the resolved metadata meets these defined semantic criteria for structured representation.
