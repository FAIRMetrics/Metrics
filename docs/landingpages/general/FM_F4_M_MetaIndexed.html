FAIR Metric - F4 - Metadata - indexed in a searchable resource

This metric evaluates whether metadata can be discovered by web-based search engines. The evaluation requires the identifier to first resolve to its metadata; then, both the identifier and the content of the retrieved metadata are used in automated search queries to determine if they are indexed and discoverable.



Principle F4 requires that metadata be registered or indexed in a searchable resource. Even if metadata is perfectly structured and contains valid identifiers, it is not "Findable" if it remains hidden from discovery services. This metric ensures that the metadata is not just "published," but actively participating in the research ecosystem. By verifying that the metadata can be found through automated queries—using both the unique identifier and the descriptive content—we confirm that the resource is effectively searchable by the machines and services that researchers rely on. Search engines are used to discover particular digital resources of interest. If the resource or its metadata are not indexed by web search engines, then this would substantially diminish the ability to find and reuse it. Thus, the ability to discover the resource should be tested using i) its identifier, ii) its text-based metadata. This metric evaluates whether the provided identifier’s metadata is discoverable by performing public, automated searches with commonly-used search engines. The evaluation assesses whether a search service can successfully locate the metadata record using either the metadata identifier or landing page content such as the record title or keywords - both commonly consumed by search engines - thereby confirming that the resource has been appropriately indexed and is accessible for automated discovery.
